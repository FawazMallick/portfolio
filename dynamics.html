System: Two-Wheeled Inverted Pendulum (TWIP)

This project models and controls a two-wheeled inverted pendulum — a wheel-mounted body that must be actively balanced. The system has four states: wheel angle (φ), wheel angular velocity (φ̇), body tilt angle (θ), and body tilt rate (θ̇), with motor torque τ as the single control input.

The equations of motion were derived symbolically using SymPy from the Lagrangian of the two-body system (wheel mass m₁ = 0.173 kg, body mass m₂ = 0.653 kg), then linearized around the upright equilibrium for LQR and Kalman filter design. For iLQR, the full nonlinear dynamics were used with numerically computed Jacobians.

Linearization Results

System A matrix (at upright equilibrium) has unstable eigenvalue at θ — confirming the pendulum is open-loop unstable. Controllability rank: 4/4. Observability rank: 4/4 (with outputs φ, θ, θ̇).

Part 1 LQR Balance Control

An LQR controller was designed by solving the continuous algebraic Riccati equation (CARE). Cost matrices Q and R were set by normalizing state and input deviations against physical limits (φmax = 2π, θmax = 0.2 rad, τmax = 5 Nm).

Optimal gain: K = [−0.796, −0.714, −51.79, −4.63]
Closed-loop response from x₀ = [0, 0, 0.1, 0] converges to upright equilibrium within ~3 seconds.

Part 2 Kalman Filter (LQE)

Because the body angle velocity θ̇ is not directly measurable in the sensor configuration (outputs: φ, θ, θ̇ measured with noise), a discrete Kalman filter was designed alongside the LQR controller to estimate the full state from noisy measurements. The system was discretized using zero-order hold (ZOH) at dt = 0.01 s, and the discrete algebraic Riccati equation was solved for the estimator gain.

Kalman gain Kkf computed. Simulation with additive process and measurement noise shows the estimated state (red dashed) tracking the true state (blue) closely over 10 seconds from x₀ = [0, 0, 0.1, 0].

Part 3 iLQR — Nonlinear Stabilization

For larger initial tilts where the linearized LQR is less reliable, iterative LQR (iLQR) was applied to the full nonlinear dynamics. iLQR alternates between a backward pass (computing locally optimal gain and feedforward terms) and a forward pass (rolling out the updated policy), converging when cost improvement falls below a threshold.

Starting from x₀ = [0, 0, 0.8, 0] with umax = 1 Nm:
iLQR converged in 7 iterations. Final cost: 127,883.
LQR (linearized) diverges from this initial condition; iLQR successfully stabilizes it.

Part 4 iLQR — Trajectory Following ("Drag Race")

iLQR was extended to trajectory following: driving the wheel to a target position of φ* = 2π radians (one full wheel revolution) while keeping the body balanced. The cost function penalizes deviation from the target state x* = [2π, 0, 0, 0] over a 10-second horizon (T = 10,000 steps at dt = 0.001 s).

Final φ: 6.2832 rad (target: 2π = 6.2832 rad) — essentially exact.
Final θ: ~−7.7 × 10⁻⁹ rad — upright maintained throughout.
Converged in 1 iteration from the LQR warm start.
